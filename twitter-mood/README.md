# Tweet Mood App

[![N|Solid](https://cldup.com/dTxpPi9lDf.thumb.png)](https://nodesource.com/products/nsolid)

Tweet Mood iOS APP.

# Dependencies used
  - RxSwift/RxCocoa - Assychronous handling of events
  - Alamofire - Offers a great to way to deal with network requests on iOS
  - Moya - Network abstraction layer on top of Alamofire
  - SDWebimage - Image caching tool
  - Swiftlint - Linter tool for swift
  - Keychain Access - Tool to manage keychain entries

# How to!

  - Clone the repository
  - You must have cocoapods installed on your mac
  - On terminal, navigate to project`s folder
  - Run on terminal:
  
```sh
$ pod install
```

# To run the project:

go to the folder: Sources/SupportFiles/EnvVars

open the file Variables.plist

On the apiVariablesTwitter Dictionary:

Enter your Twitter Api Key 

On the apiVariablesGoogle Dictionary:

Enter yout Google Api Key
